# STEER
**로컬 엔드-투-엔드 컴퓨터 제어 에이전트**

STEER는 사용자의 컴퓨터를 직접 조작하는 행동 중심 AI 에이전트입니다.
사용자가 *원하는 것*을 설명하면, STEER는 마우스, 키보드, 파일, 애플리케이션을 사용하여 필요한 작업을 계획하고 실행합니다.

## 주요 기능 (What STEER Does)
* 자연어를 실행 가능한 행동 계획으로 변환
* 마우스, 키보드 및 시스템 수준 상호 작용 제어
* 화면 상태 관찰 및 동적 행동 적응
* 범용 로컬 자동화 에이전트로 작동

## 핵심 역량 (Core Capabilities)
* 자연어 → 행동 계획
* 클릭 / 타이핑 / 열기 / 탐색 실행
* 화면 및 컨텍스트 인식
* 가드레일을 통한 안전한 실행

## 주요 사용 사례 (Typical Use Cases)
* "이 양식을 작성하고 제출해 줘"
* "이 폴더를 열고 파일 이름을 변경한 뒤 업로드해 줘"
* "이 웹사이트를 탐색해서 정보를 추출해 줘"

## 생태계 내 위치 (Position in the Ecosystem)
STEER는 STEER 생태계의 **제어 계층(Control Layer)**입니다.
단순한 추론을 넘어 *실제로 행동하는 것*에 집중합니다.

## 상태 (Status)
초기 개발 단계.
아키텍처 우선, 안전 우선 접근 방식.
