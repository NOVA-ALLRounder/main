# Steer OS Agent - 팀 소개 문서

## 🤖 이 프로젝트는 뭔가요?

**Steer**는 **Windows 자율 AI 에이전트**입니다.  
자연어로 지시하면 컴퓨터를 직접 조작해서 작업을 수행합니다.

---

## 🎯 할 수 있는 것들

### 1. 자동 UI 조작
```
"Edge 열어서 구글에 '날씨' 검색해"
"슬랙에서 #general 채널 열어"
```
→ 화면을 보고, 클릭하고, 타이핑합니다. (Windows에서는 AutoHotkey 설치 시 더 안정적)

### 2. 파일 처리
```
"Downloads 폴더에서 PDF 찾아서 내용 요약해"
"엑셀 파일 열어서 3번째 시트 데이터 읽어"
```
→ PDF, Excel, 텍스트 파일 자동 읽기

### 3. 쉘 명령 실행
```
"git status 확인해"
"npm run build 실행해"
```
→ 터미널 명령 자동 실행

### 4. 원격 제어 (텔레그램)
```
폰에서: "컴퓨터에서 슬랙 확인해"
```
→ 외출 중에도 컴퓨터 제어 가능

### 5. 워크플로우 자동화
- **반복 작업 녹화** → 매크로처럼 재생
- **n8n 워크플로우 자동 생성**
- **사용 패턴 학습** → 자동화 제안

### 6. 외부 서비스 연동
- **Notion**: 페이지 읽기/쓰기
- **Gmail**: 이메일 조회/발송
- **Calendar**: 일정 확인

---

## 🚀 시작하기 (Windows)

```powershell
# 1. 환경변수 설정
$env:OPENAI_API_KEY="sk-..."

# 2. 실행
cd apps\core
cargo run --release --bin local_os_agent

# 3. 명령어
> surf Edge 열어서 네이버 접속해
```

## 🔒 안전장치

| 기능 | 설명 |
|------|------|
| **PolicyEngine** | 위험한 작업은 승인 필요 |
| **PII 마스킹** | 민감정보 자동 익명화 |
| **쉘 타임아웃** | 30초 제한으로 무한루프 방지 |
